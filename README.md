# WEEZLY — Интеллектуальная платформа электронной коммерции

**WEEZLY** — это высокотехнологичная e-commerce платформа следующего поколения с ИИ-ассистентом, системой умных гарантий, трекером цен и полноценной административной панелью. Реализована как монолитный React/Vite/Tailwind CSS прототип с продвинутым управлением состоянием через React Context.

---

## Установка и запуск

### Предварительные требования

- **Node.js** >= 18.x — [Скачать](https://nodejs.org)
- **npm** >= 9.x (входит в состав Node.js)

### Команды для запуска

```bash
# 1. Перейти в директорию проекта
cd weezly

# 2. Установить зависимости
npm install

# 3. Запустить сервер разработки
npm run dev
```

После запуска откройте браузер по адресу: **http://localhost:5173**

### Другие команды

```bash
# Сборка для продакшн
npm run build

# Предпросмотр продакшн-сборки
npm run preview

# Проверка TypeScript
npm run typecheck

# Линтинг кода
npm run lint
```

## Функциональность

### Каталог товаров
- 16+ товаров (ноутбуки, смартфоны, умный дом, планшеты, аксессуары)
- Фильтрация по категории, бренду, диапазону цен
- Сортировка по цене, рейтингу и "смарт-баллу"

### ИИ-ассистент
- Понимает запросы на русском, беларуском и английском языках
- Извлекает категорию, бюджет и теги из текста
- "Мягкий откат" — при отсутствии точных совпадений предлагает похожие товары с объяснением

### Система отзывов
- Авторизованные пользователи могут оставлять отзывы
- Рейтинг от 1 до 5 звезд
- Мгновенный пересчет среднего рейтинга и смарт-балла при добавлении отзыва

### Трекер цен
- Пользователь устанавливает желаемую цену для товара
- Симуляция фоновых задач: каждые 35 секунд генерируются "акции"
- При достижении целевой цены: push-уведомление + письмо во "Входящих"

### Умная гарантия
- Динамический расчёт цены гарантии на основе: цены товара, категории и MTBF
- 4 уровня: Без гарантии / Базовая / Проактивная / Максимальная

### Смарт-балл
- Формула: `цена×30% + рейтинг×50% + отзывы×20%`
- Пересчитывается автоматически при добавлении отзывов

### Многоязычность (i18n)
- Языки: Русский / Беларуская / English
- Валюты: BYN / USD (курс: 1 USD = 3.25 BYN)
- Мгновенное переключение без перезагрузки

### Корзина и оформление заказа
- Выезжающая боковая панель
- Изменение количества, выбор уровня гарантии
- 4-шаговый процесс оформления с генерацией номера заказа

### Административная панель
- Дашборд с метриками (выручка, заказы, пользователи, конверсия)
- CRUD таблица товаров
- График динамики продаж
- Мониторинг инфраструктуры (PostgreSQL / Redis / Celery) в реальном времени

## Архитектура дипломного проекта

Данное приложение реализует архитектуру, описанную в дипломном проекте:

### PostgreSQL (симуляция)
- Реализован через React Context + useReducer
- Все данные (товары, отзывы, заказы, оповещения) хранятся в глобальном состоянии
- Операции CRUD выполняются через dispatch-действия

### Redis Cache (симуляция)
- Кэширование реализовано через глобальное состояние
- Товары, корзина, цены кэшируются для мгновенного доступа

### Celery Workers (симуляция)
- Реализован через setInterval в AppContext
- Каждые 35 секунд генерируются случайные скидки
- Проверяется достижение целевых цен и отправляются уведомления

### REST API (симуляция)
- Все операции выполняются синхронно через dispatch
- Состояние обновляется мгновенно без запросов к серверу
